CSS Box Model Properties

What are Box Model Properties?

The CSS Box Model is a fundamental concept that describes the rectangular boxes generated for elements in the document tree and governs how elements are rendered on the page. Understanding the box model is crucial for effective layout design in CSS.

Margin

The margin property is used to define space outside the border of the elements that surround them. The margin has 4 properties: left, right, top, and bottom. By providing the required value, we can space the elements that surround them.

We can define the margin within one line by following the format below:
margin: top right bottom left;
Alternatively, we can define each property separately as shown below:

margin-top: xx;
margin-right: xx;
margin-left: xx;
margin-bottom: xx;

Padding

The padding property defines the space between the element's border and the content. We can define the padding property within one line by following the below format

We can define the padding property within one line by following the below format:
padding : top, right, bottom, left;
Alternatively, we can define each property separately as shown below:

padding-top: xx;
padding-right: xx;
padding-left: xx;
padding-bottom: xx;

Border

The border property defines the border around the padding and content.

We can define the border property within one line by following the below format:
border : top, right, bottom, left;
Alternatively, we can define each property separately as shown below:

border-top: xx;
border-right: xx;
border-left: xx;
border-bottom: xx;

The border has one more property - "Border-radius"
The border radius property makes the border of the element in the form of round or smooth corners.
It is achieved by the following command :
Border-radius : xx;

Width & Height

The width & Height property defines the width & height of the element.

It is done as below
width: xx;
Height: xx;

Box-Shadow

This property is used to provide a 3D effect for the element. This property omits a shadow of the element behind, the position of the shadow can be adjusted by two values horizontal-offset and vertical-offset. We also have two other values one for blurring the shadow and the other to provide the color for shadow.

We can define the Box-shadow property within one line by following the below format:
box-shadow : horizontal-offset, vertical-offset, blur, desired-color;

Box-sizing

This property Determines whether padding and border are included in the element's total width and height. It has two values content-box and border-box where the content-box is default.

Using the default value, When a box is of size 200x100 then adding properties like border and padding to the box it adds them keeping the box size 200x100 as same. But when using the border-box value, the padding and border properties are alloted using the size of the box.
